# Vani

Private, native voice typing for Apple Silicon Macs.

Vani is an early-stage open-source macOS app. Hold a shortcut, speak English,
release, and insert locally transcribed text into the focused app. Audio stays on
the Mac.

## Status

Pre-alpha. The product contract and implementation plan are under review. There is
no supported build yet.

## V1 Principles

- Native Swift and Apple frameworks
- Local English transcription
- No account, telemetry, or cloud dependency
- No lost transcript when insertion fails
- Measured latency, memory, and accuracy
- Small dependency and settings surface

See [docs/PLAN.md](docs/PLAN.md) and [PRIVACY.md](PRIVACY.md).
